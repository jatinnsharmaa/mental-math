## Key Anchors

| Fact | Value |
|------|-------|
| √4   | **2** — exact |
| √9   | **3** — exact |
| √2   | ≈ **1.414** |
| √3   | ≈ **1.732** |
| ∛8   | **2** — exact |
| ∛27  | **3** — exact |
| ∛125 | **5** — exact |

> **Tip:** Square roots of non-perfect-squares fall between the roots of the nearest perfect squares.  
> e.g. √7 is between √4 = 2 and √9 = 3, closer to 3 → **≈ 2.65**
